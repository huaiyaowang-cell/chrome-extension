Poki 游戏下载器

先开启监听，再刷新页面进入游戏。
检测到游戏后资源自动下载。
游戏加载完成后点「生成 HTML」或直接停止。
若仍无下载：填写「游戏地址」或「指定下载的静态资源」后重新开始监听。

本地写盘服务

先运行 npm run poki-server。两个扩展共用下方配置；游戏按名称落在输出目录下。已存在文件会跳过。
